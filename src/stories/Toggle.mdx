# Toggle

The `Toggle` component provides a user interface for switching between states, such as on/off. It combines both `ToggleContainer` and `ToggleUI` functionalities to offer a complete toggle experience.

<Canvas of={ToggleStories.Default} />
<Controls of={ToggleStories.Default} />

### Importing Toggle

To use the `Toggle` component in your application, import it as follows:

```tsx
import { Toggle, ToggleContainer, ToggleUI, useToggleContext } from '@sendbird/uikit-react/ui/Toggle';
```

### Usage

The `Toggle` component can be customized using the following props:

- `checked`: Boolean to control the toggle state.
- `defaultChecked`: Sets the initial state.
- `disabled`: Disables the toggle.
- `reversed`: Reverses the toggle direction.
- `width`: Sets the width of the toggle.
- `animationDuration`: Controls the animation speed.
- Additional props like `name`, `id`, `ariaLabel`, and `ariaLabelledby` for accessibility.
