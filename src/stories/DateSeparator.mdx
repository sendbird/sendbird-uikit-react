# DateSeparator

The `DateSeparator` component is used to visually separate content by dates. It can display a custom text or use a default text label.

<Canvas of={DateSeparatorStories.Default} />
<Controls />

### Importing DateSeparator

To integrate the DateSeparator component into your application, use the following import statement:

```tsx
import DateSeparator from '@sendbird/uikit-react/ui/DateSeparator';
```
