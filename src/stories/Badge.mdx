# Badge

The `Badge` component is designed to display numerical values or indicators, such as unread message counts. It can be customized to show different levels of counts and can adapt to various UI themes with custom class names.
Utilize the Badge component to effectively represent numerical values or indicators within your user interface, enhancing the informational clarity of your application.

<Controls />

### Importing Badge

To use the `Badge` component in your project, import it as follows:

```tsx
import Badge from '@sendbird/uikit-react/ui/Badge';
```
