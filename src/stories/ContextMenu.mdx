# ContextMenu

The `ContextMenu` component allows for dynamic menu creation triggered by user actions. It provides a flexible way to present a list of interactive options to the user.

<Controls />

### Importing ContextMenu

To integrate the ContextMenu component into your application, use the following import statement:

```tsx
import ContextMenu, { MenuItems, MenuItem } from '@sendbird/uikit-react/ui/ContextMenu';
```
