# GroupChannel Component

The `GroupChannel` component implements the UI for a group channel.
<br/> This component allows you to display messages in a chat group, send messages, and interact with the channel in various ways.

<Canvas of={GroupChannelStories.Default} />

To integrate the Avatar component into your project, use the following import statement:
```tsx
import { GroupChannel } from '@sendbird/uikit-react/GroupChannel';
import { GroupChannelProvider, useGroupChannelContext } from '@sendbird/uikit-react/GroupChannel/context';
```
