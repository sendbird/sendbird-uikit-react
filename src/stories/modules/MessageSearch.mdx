# MessageSearch Component
The `MessageSearch` component supports searching for desired messages within the channel.
<br/>It enables searching for plain text messages or file names.

To integrate the Avatar component into your project, use the following import statement:
```tsx
import { MessageSearch } from '@sendbird/uikit-react/MessageSearch';
import { MessageSearchProvider, useMessageSearchContext } from '@sendbird/uikit-react/MessageSearch/context';
```
