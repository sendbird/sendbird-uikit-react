# Button

The `Button` component is a versatile element for triggering actions. It supports various sizes, types, disabled states, and can include custom labels. This component is fundamental for creating interactive and accessible user interfaces.

<Canvas of={ButtonStories.Primary} />
<Controls />

### Importing Button

To use the Button component in your project, add the following import statement:

```tsx
import Button from '@sendbird/uikit-react/ui/Button';
```
