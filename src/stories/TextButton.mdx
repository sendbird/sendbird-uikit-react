# TextButton

The `TextButton` component is a simple UI element that allows for clickable text elements. It supports customization such as color, disabled state, and the option to disable the underline.

<Canvas of={TextButtonStories.Default} />
<Controls of={TextButtonStories.Default} />

### Importing TextButton

To integrate the `TextButton` component into your application, use the following import statement:

```tsx
import TextButton from '@sendbird/uikit-react/ui/TextButton';
```
