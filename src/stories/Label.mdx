The `Label` component is utilized to render text within the application, supporting various typography styles and color schemes.

### Importing Label

To use the `Label` component in your project, add the following import statement:

```tsx
import Label, { LabelTypography, LabelColors } from '@sendbird/uikit-react/ui/Label';
```
