# Tooltip and TooltipWrapper

`Tooltip` and `TooltipWrapper` components provide text labels that help explain the function of a button or other user interface action.

<Canvas of={TooltipStories.Default} />
<Controls />

### Importing Tooltip and TooltipWrapper

To use these components in your application, import them as follows:

```tsx
import Tooltip from '@sendbird/uikit-react/ui/Tooltip';
import TooltipWrapper from '@sendbird/uikit-react/ui/TooltipWrapper';
```
