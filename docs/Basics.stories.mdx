# Basics

With Sendbird UI Kit React, we will handle the life cycle of the SDK

* SendbirdProvider - The context provider for SDK component
* withSendbird - The Higher Order Component if you want to handle component state
* ChannelList - A ChannelList UI component
* Channel - A UI Component where conversations happen
* ChannelSetting - A component to handle the settings of a given channel

* App - is a full fledged app component made by combining the above components so that you dont have

Dont forget to import stylesheet from the repo too
